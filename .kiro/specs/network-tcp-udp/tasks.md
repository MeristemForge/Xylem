# 实现计划: network-tcp-udp

## 概述

基于设计文档，为 Xylem C 工具库实现非阻塞 TCP 和 UDP 网络模块。按自底向上顺序构建：先扩展 ringbuf peek API，再实现地址工具，然后 TCP（帧解析 → 连接 → 服务端），最后 UDP，逐步集成并测试。
