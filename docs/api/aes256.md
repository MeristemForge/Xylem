# AES-256

`#include <xylem/xylem-aes256.h>`

AES-256 对称加密，支持 CTR 和 CBC 模式。IV 由内部 CSPRNG 自动生成并前缀到输出。

---

## 类型

### xylem_aes256_t {#xylem_aes256_t}

不透明类型，表示一个 AES-256 上下文。通过 [`xylem_aes256_create()`](#xylem_aes256_create) 获得。上下文可复用于多次加密/解密操作。

---

## 大小计算

### xylem_aes256_ctr_encrypt_size {#xylem_aes256_ctr_encrypt_size}

```c
size_t xylem_aes256_ctr_encrypt_size(size_t len);
```

计算 CTR 加密输出所需的缓冲区大小。

| 参数 | 类型 | 说明 |
|------|------|------|
| `len` | `size_t` | 明文长度 |

**返回值：** 所需输出缓冲区大小。

---

### xylem_aes256_ctr_decrypt_size {#xylem_aes256_ctr_decrypt_size}

```c
size_t xylem_aes256_ctr_decrypt_size(size_t len);
```

计算 CTR 解密输出的最大大小。

| 参数 | 类型 | 说明 |
|------|------|------|
| `len` | `size_t` | 密文长度（含 IV 前缀） |

**返回值：** 最大明文输出大小，`len` 过小返回 0。

---

### xylem_aes256_cbc_encrypt_size {#xylem_aes256_cbc_encrypt_size}

```c
size_t xylem_aes256_cbc_encrypt_size(size_t len);
```

计算 CBC 加密输出所需的缓冲区大小（含 IV 和 PKCS7 padding）。

| 参数 | 类型 | 说明 |
|------|------|------|
| `len` | `size_t` | 明文长度 |

**返回值：** 所需输出缓冲区大小。

---

### xylem_aes256_cbc_decrypt_size {#xylem_aes256_cbc_decrypt_size}

```c
size_t xylem_aes256_cbc_decrypt_size(size_t len);
```

计算 CBC 解密输出的最大大小。实际输出可能因 PKCS7 unpadding 而更小。

| 参数 | 类型 | 说明 |
|------|------|------|
| `len` | `size_t` | 密文长度（含 IV 前缀） |

**返回值：** 最大明文输出大小，`len` 过小返回 0。

## 生命周期

### xylem_aes256_create {#xylem_aes256_create}

```c
xylem_aes256_t* xylem_aes256_create(const uint8_t key[32]);
```

创建 AES-256 上下文并执行密钥扩展。上下文可复用于多次加密/解密。

| 参数 | 类型 | 说明 |
|------|------|------|
| `key` | `const uint8_t[32]` | 32 字节密钥 |

**返回值：** 上下文句柄，失败返回 `NULL`。

---

### xylem_aes256_destroy {#xylem_aes256_destroy}

```c
void xylem_aes256_destroy(xylem_aes256_t* ctx);
```

销毁上下文并清零敏感数据。

| 参数 | 类型 | 说明 |
|------|------|------|
| `ctx` | [`xylem_aes256_t*`](#xylem_aes256_t) | 上下文句柄 |

---

## CTR 模式

### xylem_aes256_ctr_encrypt {#xylem_aes256_ctr_encrypt}

```c
int xylem_aes256_ctr_encrypt(xylem_aes256_t* ctx,
                             const uint8_t* src, size_t slen,
                             uint8_t* dst, size_t dlen);
```

AES-256-CTR 加密。内部生成随机 16 字节 IV 并前缀到输出。

| 参数 | 类型 | 说明 |
|------|------|------|
| `ctx` | [`xylem_aes256_t*`](#xylem_aes256_t) | 上下文 |
| `src` | `const uint8_t*` | 明文输入 |
| `slen` | `size_t` | 输入长度（任意长度） |
| `dst` | `uint8_t*` | 输出缓冲区（IV + 密文） |
| `dlen` | `size_t` | 输出缓冲区大小 |

**返回值：** 写入 `dst` 的字节数，失败返回 -1。

---

### xylem_aes256_ctr_decrypt {#xylem_aes256_ctr_decrypt}

```c
int xylem_aes256_ctr_decrypt(xylem_aes256_t* ctx,
                             const uint8_t* src, size_t slen,
                             uint8_t* dst, size_t dlen);
```

AES-256-CTR 解密。从输入前端读取 IV。

| 参数 | 类型 | 说明 |
|------|------|------|
| `ctx` | [`xylem_aes256_t*`](#xylem_aes256_t) | 上下文 |
| `src` | `const uint8_t*` | 输入（IV + 密文） |
| `slen` | `size_t` | 输入长度 |
| `dst` | `uint8_t*` | 明文输出缓冲区 |
| `dlen` | `size_t` | 输出缓冲区大小 |

**返回值：** 写入 `dst` 的字节数，失败返回 -1。

---

## CBC 模式

### xylem_aes256_cbc_encrypt {#xylem_aes256_cbc_encrypt}

```c
int xylem_aes256_cbc_encrypt(xylem_aes256_t* ctx,
                             const uint8_t* src, size_t slen,
                             uint8_t* dst, size_t dlen);
```

AES-256-CBC 加密，自动 PKCS7 padding。内部生成随机 IV 并前缀到输出。

| 参数 | 类型 | 说明 |
|------|------|------|
| `ctx` | [`xylem_aes256_t*`](#xylem_aes256_t) | 上下文 |
| `src` | `const uint8_t*` | 明文输入 |
| `slen` | `size_t` | 输入长度（任意长度） |
| `dst` | `uint8_t*` | 输出缓冲区（IV + 填充密文） |
| `dlen` | `size_t` | 输出缓冲区大小 |

**返回值：** 写入 `dst` 的字节数，失败返回 -1。

---

### xylem_aes256_cbc_decrypt {#xylem_aes256_cbc_decrypt}

```c
int xylem_aes256_cbc_decrypt(xylem_aes256_t* ctx,
                             const uint8_t* src, size_t slen,
                             uint8_t* dst, size_t dlen);
```

AES-256-CBC 解密，自动 PKCS7 unpadding。从输入前端读取 IV。

| 参数 | 类型 | 说明 |
|------|------|------|
| `ctx` | [`xylem_aes256_t*`](#xylem_aes256_t) | 上下文 |
| `src` | `const uint8_t*` | 输入（IV + 填充密文） |
| `slen` | `size_t` | 输入长度 |
| `dst` | `uint8_t*` | 明文输出缓冲区 |
| `dlen` | `size_t` | 输出缓冲区大小 |

**返回值：** 写入 `dst` 的字节数，失败返回 -1（`dlen` 不足、padding 无效或输入长度错误）。
